# Codeviz Local

로컬 코드베이스 구조 분석기 (웹 뷰어 제외, CLI + GUI).

## Quick Start (CLI)

```powershell
cd "E:\code visualize\codeviz-local"
$env:PYTHONPATH = "E:\code visualize\codeviz-local"
python -m apps.cli.main --config .\configs\default.yaml
```

## Quick Start (GUI)

```powershell
cd "E:\code visualize\codeviz-local"
$env:PYTHONPATH = "E:\code visualize\codeviz-local"
python -m apps.desktop.gui
```

## Dependency

```powershell
python -m pip install -r .\requirements.txt
```
